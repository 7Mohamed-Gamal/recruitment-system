# 🏢 Recruitment System - Orient

نظام إدارة التوظيف لشركة Orient، مبني على ClickUp مع أتمتة كاملة باستخدام Super Agents وربط مع Google Calendar و Gmail.

---

## 📋 نظرة عامة

نظام متكامل لإدارة عملية التوظيف من استلام طلب المرشح حتى التعيين أو الرفض، مع:
- تسجيل تلقائي للمرشحين
- جدولة المقابلات على Google Calendar
- إرسال إيميلات تأكيد تلقائية
- تذكيرات يومية بالمقابلات القادمة
- Board View لتتبع مراحل التوظيف

## 🤖 Super Agents

### Recruiting Rami
| الخاصية | التفاصيل |
|---------|----------|
| الوظيفة | استقبال وتسجيل طلبات التوظيف |
| Triggers | DM, @mention |
| القدرات | إنشاء تاسك + ملء الحقول + إنشاء حدث Google Calendar + إرسال إيميل تأكيد |

**سير العمل:**
1. يستقبل بيانات المرشح عبر رسالة
2. يتحقق من البيانات الأساسية (الاسم، الوظيفة، الهاتف)
3. ينشئ تاسك في قائمة التوظيف
4. يملأ الحقول المخصصة
5. ينشئ حدث Google Calendar (لو في تاريخ مقابلة)
6. يبعت إيميل تأكيد للمرشح (لو في بريد إلكتروني)

---

### Reminder Rania
| الخاصية | التفاصيل |
|---------|----------|
| الوظيفة | تذكير بمواعيد المقابلات |
| Triggers | DM, @mention |
| Schedule | يومياً الساعة 9:00 صباحاً |
| القدرات | بحث عن مقابلات بكرة + كومنت تذكيري + mention للمسؤول |

**سير العمل:**
1. كل يوم الساعة 9 صباحاً تبحث عن مقابلات الغد
2. تبعت كومنت على كل تاسك فيه مقابلة
3. تعمل mention للمسؤول عن المقابلة
